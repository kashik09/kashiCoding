# Technical Consulting Spec

## Features
- Code review and refactor advice.
- Architecture and stack consultation.
- Performance and security audits.
- Database optimization guidance.
- Team training sessions.

## Constraints
- Engagement is time-boxed.
- Access to codebase required.
- Findings delivered as reports.

## Assumptions
- Stakeholders are available for interviews.
- Teams can implement recommendations.
- Access is granted to necessary systems.

## Open questions
- Preferred engagement cadence?
- Do we need an on-site workshop?
- What success metrics matter most?
