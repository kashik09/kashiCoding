# Authentication & User Management Boilerplate

## What it is
- Authentication boilerplate built on NextAuth.js.
- Role-based access control and protected routes.
- User profile management and reset flows.

## Who it is for
- Teams building apps that need auth fast.
- Startups needing roles and user management.
- Developers who want a proven auth baseline.

## What's included
- NextAuth.js integration with providers.
- Role-based permissions and guards.
- Email and social sign-in flows.
- Password reset and session handling.
- User profile UI scaffolds.

## What it is NOT
- A hosted identity provider.
- A full CRM or user analytics suite.
- A replacement for SSO providers.
