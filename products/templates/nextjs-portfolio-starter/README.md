# Next.js Portfolio Starter

## What it is
- Next.js 14 App Router portfolio template.
- JSON-based CMS with admin editing tools.
- Theme system with screenshot capture hooks.

## Who it is for
- Developers who want a strong portfolio baseline.
- Freelancers who need editable content and themes.
- Designers handing off to a developer.

## What's included
- Next.js + TypeScript scaffolding.
- JSON content schemas and API routes.
- Admin editor UI and auth-ready structure.
- Multi-theme styles and toggles.
- Screenshot capture workflow notes.

## What it is NOT
- A hosted SaaS or managed service.
- Custom design or copywriting.
- A full ecommerce system.
