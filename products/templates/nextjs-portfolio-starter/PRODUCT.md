# Next.js Portfolio Starter Spec

## Features
- Next.js 14 App Router with TypeScript.
- JSON-based CMS and admin editor.
- Multi-theme system and responsive UI.
- Screenshot capture workflow support.
- SEO-friendly defaults.

## Constraints
- App Router only; no Pages Router support.
- File-based content storage (no database).
- Requires Node 20 for local development.

## Assumptions
- Content is stored and versioned in the repo.
- Deployment on Vercel or similar hosting.
- Users will customize branding and copy.
