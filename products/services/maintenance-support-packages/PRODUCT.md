# Maintenance & Support Packages Spec

## Features
- Tiered monthly hour allocations.
- Security and dependency updates.
- Performance monitoring.
- Backup management.
- Priority support.

## Constraints
- Hours capped by package tier.
- Scope excludes major new features.
- Requires access to hosting and repos.

## Assumptions
- Clients provide access and approvals.
- Work requests are submitted via ticketing.
- Reporting cadence is monthly.

## Open questions
- What are the target response times per tier?
- Which monitoring tools are preferred?
- How should emergency requests be handled?
