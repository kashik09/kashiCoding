# JSON-Based CMS Kit

## What it is
- JSON file CMS with API routes and editors.
- Type-safe content models for multiple pages.
- Git-friendly content workflow.

## Who it is for
- Teams that want a CMS without a database.
- Developers who prefer file-based content.
- Projects that need simple editing tools.

## What's included
- JSON storage patterns and helpers.
- REST API routes for content operations.
- Editor components and previews.
- Versioning and timestamp metadata.
- Content type examples.

## What it is NOT
- A hosted CMS or headless service.
- Media hosting or asset CDN.
- Complex workflow or approvals.
