# JSON API Builder

## What it is
- Visual builder for JSON schemas.
- Auto-generated REST APIs and docs.
- Authentication and versioning options.

## Who it is for
- Product teams needing quick APIs.
- Non-developers building internal tools.
- Developers who want a schema-driven workflow.

## What's included
- Schema builder UI concept.
- API generation engine outline.
- Docs generator and testing playground plan.
- Authentication options (API keys, JWT).
- Export options (OpenAPI, Postman).

## What it is NOT
- A full database hosting platform.
- A replacement for custom APIs in all cases.
- A general purpose low-code platform.
